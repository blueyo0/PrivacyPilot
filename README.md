# PrivacyPilot: An LLM Agent Plugin for Privacy Protection

A locally deployed plugin integration for Agentic LLM Apps. This repo is at a very early stage of development.

## Install
```
pip install ppilot
```

## Quick Start
Just run this cmd:
```
ppilot_serve
```
